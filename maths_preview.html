What is a Determinant?

Every square matrix (a matrix where rows = columns) has a special number attached to it. That number is called its Determinant.

Think of it this way — a matrix is like a box full of numbers. The determinant is a single number you can calculate from that box. It tells you a lot about what the matrix does.

📌 Simple Definition

A Determinant is a single number calculated from a square matrix. It tells us whether the matrix has an inverse or not, and helps us solve equations. We write it as det(A) or |A|.

1×1 Matrix Determinant (Simplest case)

For a 1×1 matrix, the determinant is just the number itself.

Matrix A
5
→
det(A)
= 5

2×2 Matrix Determinant

This is the most important one for your exam. Here is how it works:

Matrix A
a
b
c
d
→
det(A) = |A|
ad − bc
📐 Formula · 2×2 Determinant
|A| = (a × d) − (b × c)

The rule is simple: multiply the diagonal going down-right (a×d), then subtract the diagonal going down-left (b×c).

✅ Solved Example — 2×2

✅ Solved Example

Find the determinant of:

3
2
1
4
1

Identify: a = 3, b = 2, c = 1, d = 4

2

Apply the formula: |A| = (a×d) − (b×c)

|A| = (3 × 4) − (2 × 1)
3

Calculate:

|A| = 12 − 2 = 10

3×3 Matrix Determinant

For a 3×3 matrix, we use the expansion along the first row. Don't worry — it looks big but it is actually just repeating the 2×2 formula 3 times!

a₁
b₁
c₁
a₂
b₂
c₂
a₃
b₃
c₃
📐 Formula · 3×3 Determinant (Expansion along Row 1)
|A| = a₁(b₂c₃ − b₃c₂) − b₁(a₂c₃ − a₃c₂) + c₁(a₂b₃ − a₃b₂)
⚡ Easy way to remember the signs

When expanding along the first row, the signs follow: + − +

First element → + (positive), Second → − (negative), Third → + (positive)

Expand along Row 1 using elements 1, 2, 3

2

+1 × det of minor (remove row 1, col 1):

+1 × [(5×9) − (6×8)] = 1 × [45 − 48] = 1 × (−3) = −3
3

−2 × det of minor (remove row 1, col 2):

−2 × [(4×9) − (6×7)] = −2 × [36 − 42] = −2 × (−6) = +12
4

+3 × det of minor (remove row 1, col 3):

+3 × [(4×8) − (5×7)] = 3 × [32 − 35] = 3 × (−3) = −9
5

Add all three:

|A| = −3 + 12 + (−9) = 0
💡 Note

When determinant = 0, the matrix is called a Singular Matrix. It has no inverse!

Singular vs Non-Singular Matrix

The value of the determinant tells us something very important about the matrix:

Type Condition Has Inverse? Example
Singular Matrix det(A) = 0 ❌ No det = 0
Non-Singular Matrix det(A) ≠ 0 ✅ Yes det = 10
🎯

DDCET Exam Tip: This is asked almost every year! If they ask "Find the value of x so that the matrix is singular" — just set det(A) = 0 and solve for x. Simple!

Practice Questions

📝 Practice Q1

Find the determinant of the matrix A = [[2, 5], [1, 3]]

Answer: |A| = (2×3) − (5×1) = 6 − 5 = 1
📝 Practice Q2 (From DDCET 2024)

For what value of x is the matrix [[x, 2], [3, 6]] a singular matrix?

Set det = 0: (6x) − (2×3) = 0 → 6x − 6 = 0 → x = 1

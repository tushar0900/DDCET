Part A — Determinants

🔰 What is a Determinant? (Zero thi shuru)

Ek square matrix hoy chhe — jema rows ane columns ni count barabar hoy chhe. Dareek square matrix sathe ek khaas number associated hoy chhe, which we call its Determinant.

It is written as |A| or det(A). Think of it as a single number that carries the "essence" of the matrix. It is used to:

  • Check if a system of equations has a solution
  • Find the inverse of a matrix
  • Calculate area of triangles using coordinates
💡
Key RuleDeterminant is only defined for square matrices (2×2, 3×3, etc.). You cannot find determinant of a 2×3 or 3×2 matrix!
2×2 Determinant (Second Order)

Formula & Method — 2×2

For a 2×2 matrix with elements a, b, c, d:

2nd Order Determinant
|A| = |a b| = a×d − b×c |c d| Think: Diagonal down (×) MINUS Diagonal up (×)
📝 Example 1 — Find |A| where A = [[5, 3], [2, 4]]
1
Write the determinant: |5 3| and |2 4|
2
Apply formula: |A| = (5×4) − (3×2)
3
= 20 − 6 = 14
📝 Example 2 — Find |A| where A = [[−2, 1], [3, 5]]
1
Main diagonal: (−2)×5 = −10
2
Off diagonal: 1×3 = 3
3
|A| = −10 − 3 = −13
📝 Example 3 — If |3 x / 2 4| = 0, find x
1
Apply formula: 3×4 − x×2 = 0
2
12 − 2x = 0
3
x = 6
3×3 Determinant (Third Order)

Expansion Along First Row

3×3 determinant expanded along Row 1 using cofactors. Signs are: + − +

3rd Order — Row 1 Expansion
|a₁ b₁ c₁| |a₂ b₂ c₂| = a₁|b₂ c₂| − b₁|a₂ c₂| + c₁|a₂ b₂| |a₃ b₃ c₃| |b₃ c₃| |a₃ c₃| |a₃ b₃| = a₁(b₂c₃−b₃c₂) − b₁(a₂c₃−a₃c₂) + c₁(a₂b₃−a₃b₂)
🎯
Sign PatternAlways remember: Row 1 signs are + − +. Middle element b₁ has MINUS sign. Bhulsho nathi — 3×3 ni calculation ma aa common mistake che!
📝 Example 1 — Find |2 1 3 / 0 4 1 / 5 2 0|
1
Expand along R1: 2×|4 1/2 0| − 1×|0 1/5 0| + 3×|0 4/5 2|
2
= 2(4×0−1×2) − 1(0×0−1×5) + 3(0×2−4×5)
3
= 2(0−2) − 1(0−5) + 3(0−20)
4
= 2(−2) − 1(−5) + 3(−20) = −4 + 5 − 60 = −59
📝 Example 2 — Find |1 2 3 / 4 5 6 / 7 8 9|
1
= 1(5×9−6×8) − 2(4×9−6×7) + 3(4×8−5×7)
2
= 1(45−48) − 2(36−42) + 3(32−35)
3
= 1(−3) − 2(−6) + 3(−3) = −3+12−9 = 0
Properties of Determinants — Exam Gold!

⚡ All Properties with Examples

Properties — All Must Know
P1: |Aᵀ| = |A| (transpose doesn't change determinant) P2: If any row/column is ALL ZEROS → |A| = 0 P3: Two IDENTICAL rows/columns → |A| = 0 P4: If row₁ = k × row₂ (proportional) → |A| = 0 P5: Swapping two rows → |A| becomes −|A| P6: Multiplying any ONE row by k → |kA_row| = k × |A| P7: For n×n matrix: |kA| = kⁿ × |A| P8: |AB| = |A| × |B| P9: |A⁻¹| = 1/|A|
🎯
DDCET Fastest Tricks 1. Identical rows or columns → answer is 0 directly! 2. |2A| for 3×3 = 2³|A| = 8|A| 3. |A⁻¹| = 1/|A| — very frequent MCQ!
📝 Property MCQ Example — |A| = 5, find |adj(A)| for 3×3
1
Formula: |adj(A)| = |A|^(n−1) where n = order
2
n = 3, so |adj(A)| = |A|^(3−1) = 5² = 25
Minors and Cofactors

Minor (Mᵢⱼ) and Cofactor (Cᵢⱼ)

Minor Mᵢⱼ

Delete the i-th row and j-th column. The determinant of the remaining smaller matrix is called the Minor of element at position (i,j).

Cofactor Formula
Cᵢⱼ = (−1)^(i+j) × Mᵢⱼ Sign pattern for 3×3: + − + − + − + − +
📝 Example — Find cofactor C₁₂ of |1 2 3 / 4 5 6 / 7 8 9|
1
i=1, j=2 → Delete row 1, column 2
2
Remaining: |4 6| → M₁₂ = 4×9 − 6×7 = 36−42 = −6 |7 9|
3
C₁₂ = (−1)^(1+2) × (−6) = (−1)³ × (−6) = (−1)(−6) = 6
Part B — Matrices

🔰 What is a Matrix? (Bilkul basic)

Ek matrix ek rectangular arrangement of numbers (or expressions) in rows and columns. It is written inside square brackets [ ].

Matrix Notation
A = [aᵢⱼ] where i = row number, j = column number Order (Size) = m × n (m rows, n columns) Example: A = [1 2 3] ← This is 2×3 matrix [4 5 6] (2 rows, 3 columns)

Element a₂₃ means: row 2, column 3. In the above example, a₂₃ = 6.

Types of Matrices — All Must Know

Complete List of Matrix Types

Row Matrix

Only 1 row. Example: [1 3 5]
Order: 1×n

Column Matrix

Only 1 column. Order: m×1
Example: [2 / 4 / 6] (vertical)

Square Matrix

Rows = Columns. Order n×n
Example: 2×2, 3×3

Null / Zero Matrix (O)

All elements = 0.
[0 0 / 0 0]

Identity Matrix (I)

Diagonal = 1, rest = 0. I × A = A × I = A

Diagonal Matrix

All non-diagonal elements = 0. Diagonal can be anything.

Scalar Matrix

Diagonal matrix where all diagonal elements are equal.

Symmetric Matrix

Aᵀ = A. Element aᵢⱼ = aⱼᵢ

Skew-Symmetric

Aᵀ = −A. Diagonal elements must be 0.

Singular Matrix

|A| = 0. Inverse does NOT exist.

Matrix Addition & Scalar Multiplication

Addition Rules

Addition Conditions & Formula
Condition: Both matrices must have SAME ORDER (A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ (add corresponding elements) Scalar: kA means multiply EVERY element by k Properties: A + B = B + A (commutative) (A+B)+C = A+(B+C) (associative) A + O = A (O = null matrix)
📝 Example — If A = [[1,2],[3,4]] and B = [[5,6],[7,8]], find A+B and 2A
1
A+B = [[1+5, 2+6],[3+7, 4+8]] = [[6,8],[10,12]]
2
2A = [[2×1, 2×2],[2×3, 2×4]] = [[2,4],[6,8]]
Matrix Multiplication

Multiplication — Most Important!

Condition & Method
Condition: Columns of A = Rows of B If A is (m×n) and B is (n×p) → result C is (m×p) cᵢⱼ = Σ aᵢₖ × bₖⱼ (row of A × column of B, then sum)
⚠️
Not Commutative! AB ≠ BA in general. But (AB)C = A(BC) (associative). Also AI = IA = A.
📝 Example — A = [[1,2],[3,4]], B = [[5,6],[7,8]], find AB
1
C₁₁ = Row1 of A × Col1 of B = 1×5 + 2×7 = 5+14 = 19
2
C₁₂ = Row1 of A × Col2 of B = 1×6 + 2×8 = 6+16 = 22
3
C₂₁ = Row2 of A × Col1 of B = 3×5 + 4×7 = 15+28 = 43
4
C₂₂ = Row2 of A × Col2 of B = 3×6 + 4×8 = 18+32 = 50
5
AB = [[19, 22],[43, 50]]
Transpose of a Matrix

Transpose — Rows Become Columns

To find Aᵀ: 1st row becomes 1st column, 2nd row becomes 2nd column, and so on.

Transpose Properties
(Aᵀ)ᵀ = A (A + B)ᵀ = Aᵀ + Bᵀ (AB)ᵀ = Bᵀ × Aᵀ ← Order reverses! (kA)ᵀ = k·Aᵀ
📝 Example — Find Aᵀ of A = [[1,2,3],[4,5,6]]
1
A is 2×3. Aᵀ will be 3×2.
2
Aᵀ = [[1,4],[2,5],[3,6]] — rows of A became columns
Adjoint of a Matrix

Adjoint (adj A) — Step by Step

Adjoint = Transpose of the Cofactor Matrix

📝 Find adj(A) for A = [[1,2],[3,4]]
1
Find cofactors: C₁₁ = +4, C₁₂ = −3, C₂₁ = −2, C₂₂ = +1
2
Cofactor matrix = [[4, −3],[−2, 1]]
3
adj(A) = Transpose = [[4, −2],[−3, 1]]
💡
2×2 Shortcut for adj(A) For [[a,b],[c,d]] → adj(A) = [[d,−b],[−c,a]]. Just swap diagonals, change signs of off-diagonal!
Inverse of a Matrix

A⁻¹ — How to Find

Inverse Formula
A⁻¹ = adj(A) / |A| Conditions: • |A| ≠ 0 (matrix must be non-singular) • Verify: A × A⁻¹ = A⁻¹ × A = I For 2×2 shortcut: If A = [[a,b],[c,d]] A⁻¹ = 1/(ad−bc) × [[d, −b],[−c, a]]
📝 Example — Find A⁻¹ for A = [[2,1],[5,3]]
1
|A| = 2×3 − 1×5 = 6−5 = 1
2
adj(A) = [[3,−1],[−5,2]] (swap diagonals, negate off-diag)
3
A⁻¹ = (1/1) × [[3,−1],[−5,2]] = [[3,−1],[−5,2]]
4
Verify: A × A⁻¹ = [[2×3+1×(−5), 2×(−1)+1×2],[...]] = [[1,0],[0,1]] = I ✓
Solving Linear Equations — Cramer's Rule

Cramer's Rule — Most Asked in DDCET!

For two equations: a₁x + b₁y = c₁ and a₂x + b₂y = c₂

Cramer's Rule Formulas
D = |a₁ b₁| (coefficient matrix determinant) |a₂ b₂| D₁ = |c₁ b₁| (replace column 1 with constants) |c₂ b₂| D₂ = |a₁ c₁| (replace column 2 with constants) |a₂ c₂| x = D₁/D y = D₂/D If D = 0 and D₁=D₂=0 → Infinite solutions If D = 0 but D₁≠0 or D₂≠0 → No solution
📝 Example — Solve: 2x + 3y = 8, 4x + y = 6
1
D = |2 3 / 4 1| = 2×1 − 3×4 = 2−12 = −10
2
D₁ = |8 3 / 6 1| = 8×1 − 3×6 = 8−18 = −10
3
D₂ = |2 8 / 4 6| = 2×6 − 8×4 = 12−32 = −20
4
x = D₁/D = −10/−10 = 1, y = D₂/D = −20/−10 = 2

🔰 What is an Angle? Converting Degree ↔ Radian

Angle means the amount of rotation from one ray to another. We measure it in Degrees (°) or Radians (rad). DDCET ma conversion direct puchhay chhe!

Conversion Formula — Must Memorize
π radians = 180° ← This is the KEY! Degree → Radian: Multiply by π/180 Radian → Degree: Multiply by 180/π Quick Reference (Yaad karo!): 30° = π/6 45° = π/4 60° = π/3 90° = π/2 120° = 2π/3 135° = 3π/4 150° = 5π/6 180° = π 225° = 5π/4 270° = 3π/2 315° = 7π/4 360° = 2π
📝 Example — Convert 150° to radians and 5π/6 to degrees
1

2
5π/6 to degree: (5π/6) × (180/π) = 5×30 = 150° ✅
3
225° to radian (2025 paper Q65): 225 × π/180 = 5π/4 ✅

Pythagorean Identities

Three Fundamental Identities — Exam ma 100% aave chhe!

Pythagorean Identities — Must Know ALL!
sin²θ + cos²θ = 1 → sin²θ = 1 − cos²θ | cos²θ = 1 − sin²θ 1 + tan²θ = sec²θ → sec²θ − tan²θ = 1 | (secθ+tanθ)(secθ−tanθ) = 1 1 + cot²θ = cosec²θ → cosec²θ − cot²θ = 1 Derived: sin²θ = (1−cos2θ)/2 cos²θ = (1+cos2θ)/2

180°±θ and 360°±θ: Function STAYS SAME. Sign from ASTC.
Yaad rakhva nu: "Half (90°, 270°) ma function badle, full (180°, 360°) ma same rahe!"

🔰 Scalar vs Vector — Basic Difference

Scalar Quantity

Has only magnitude (size/number). Direction hotu nathi.

Examples: mass (5 kg), temperature (30°C), speed (60 km/h), time, distance, area, volume

Vector Quantity

Has both magnitude AND direction. Banne jaruri chhe.

Examples: force (10N East), velocity (60 km/h North), displacement, acceleration, momentum

Vector Notation
Vector AB written as: →AB or a⃗ or a (bold) In component form: a⃗ = a₁î + a₂ĵ + a₃k̂ î = unit vector along X-axis ĵ = unit vector along Y-axis k̂ = unit vector along Z-axis Example: a⃗ = 3î + 4ĵ − 2k̂ means 3 in X, 4 in Y, −2 in Z direction
Types of Vectors

Magnitude = 0, no direction. Written as 0⃗

Magnitude of a Vector

How to Find Magnitude (Length) of a Vector

Magnitude Formula
If a⃗ = a₁î + a₂ĵ + a₃k̂, then: |a⃗| = √(a₁² + a₂² + a₃²) In 2D: |a⃗| = √(a₁² + a₂²)
📝 Examples

2
b⃗ = 1î + 2ĵ + 2k̂ → |b⃗| = √(1+4+4) = √9 = 3
3
c⃗ = 2î − 3ĵ + 6k̂ → |c⃗| = √(4+9+36) = √49 = 7

Unit vector = Vector divided by its magnitude.

â = a⃗ / |a⃗| (magnitude = 1 after dividing)

Vector Addition and Subtraction

Adding and Subtracting Vectors

Component-wise Addition/Subtraction
If a⃗ = a₁î+a₂ĵ+a₃k̂ and b⃗ = b₁î+b₂ĵ+b₃k̂ a⃗ + b⃗ = (a₁+b₁)î + (a₂+b₂)ĵ + (a₃+b₃)k̂ a⃗ − b⃗ = (a₁−b₁)î + (a₂−b₂)ĵ + (a₃−b₃)k̂ Properties: a⃗ + b⃗ = b⃗ + a⃗ (commutative) (a⃗+b⃗)+c⃗ = a⃗+(b⃗+c⃗) (associative)
Dot Product (Scalar Product)

Dot Product — Result is a SCALAR Number

Dot Product Formulas
a⃗ · b⃗ = |a⃗||b⃗| cosθ (θ = angle between vectors) Component form: a⃗ · b⃗ = a₁b₁ + a₂b₂ + a₃b₃ Special values: î·î = ĵ·ĵ = k̂·k̂ = 1 î·ĵ = ĵ·k̂ = k̂·î = 0 Perpendicular: a⃗·b⃗ = 0 (θ=90°, cos90°=0) Parallel: a⃗·b⃗ = |a⃗||b⃗| (θ=0°, cos0°=1)
Finding Angle Between Vectors
cosθ = (a⃗ · b⃗) / (|a⃗| × |b⃗|)

🔰 What is Slope? (Basic Concept)

Slope (m) tells us how steep a line is — how much it goes up or down per unit horizontal distance. It is also called the gradient.

Slope Formulas
From two points (x₁,y₁) and (x₂,y₂): m = y₂ − y₁x₂ − x₁ From angle with X-axis: m = tan θ From general equation ax+by+c=0: m = −a/b From slope-intercept y = mx+c: m is the coefficient of x
📝 Examples
1
Points (2,3) and (5,9): m = (9−3)/(5−2) = 6/3 = 2
2
3x+4y−8=0: m = −3/4 = −0.75
3
Horizontal line y=5: m = 0
4
Vertical line x=3: m = ∞ (undefined)

Parallel and Perpendicular Lines

Conditions
Parallel lines: m₁ = m₂ (same slope, different intercepts) Perpendicular lines: m₁ × m₂ = −1 i.e., m₂ = −1/m₁ (negative reciprocal) Coincident lines: m₁ = m₂ AND c₁ = c₂
🎯
DDCET Favorite! If slope of a line is 3, then slope of perpendicular = −1/3. If slope = 2/5, perpendicular slope = −5/2. Just flip and change sign!

y − y₁ = m(x − x₁) (passes through (x₁,y₁) with slope m)

(y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)

2

x/a + y/b = 1 (a=x-intercept, b=y-intercept)

1

Standard Forms of Circle
Centre at ORIGIN (0,0): x² + y² = r² Centre at (h,k), radius r: (x−h)² + (y−k)² = r² GENERAL Form: x² + y² + 2gx + 2fy + c = 0 Centre = (−g, −f) Radius = √(g² + f² − c) Condition: g²+f²−c > 0 (real circle)

🔰 Ordered Pair and Relation

An ordered pair (a, b) means a comes first, b comes second. Order matters: (2,3) ≠ (3,2).

A relation R from set A to set B is any subset of A×B (Cartesian product). In simple words — any rule that connects elements of A to elements of B.

Cartesian Product
A = {1,2}, B = {3,4} A×B = {(1,3), (1,4), (2,3), (2,4)} ← All possible pairs

Function — Special Type of Relation

A function f: A → B means every element of A has EXACTLY ONE image in B. No element is left out, no element maps to two different values.

Domain, Codomain, Range
Domain (Dƒ) = Set A (all INPUTS — must all be used) Codomain (CoDƒ) = Set B (all possible outputs) Range (Rƒ) = {actual outputs that f produces} Range ⊆ Codomain always (range can't be bigger than codomain)

Is it a Function? — How to Check

  • Every element of A must have an arrow → if any element has NO arrow, function nathi.
  • Each element of A must go to ONLY ONE element in B → if any element has 2 arrows, function nathi.
  • Elements of B can receive 0, 1, or many arrows — that's fine!
Types of Functions — All Types

One-One (Injective) and Onto (Surjective)

One-One (Injective)

Different inputs give different outputs. f(a) = f(b) ⟹ a = b

Many-One

Two or more inputs can give SAME output. NOT one-one.

Onto (Surjective)

Every element of B is used. Range = Codomain.

Into

At least one element of B is NOT used. Range ⊊ Codomain.

Bijection: A function that is BOTH one-one AND onto. It has a perfect inverse.

Standard Types of Functions

Constant Function

f(x) = k (same output always) Graph = horizontal line

Identity Function

f(x) = x (output = input) Graph = line through origin at 45°

Modulus / Absolute Value

|x| = x if x ≥ 0 |x| = −x if x < 0 |5|=5, |−5|=5

Polynomial Function

f(x) = aₙxⁿ+...+a₁x+a₀ e.g. 3x²+2x−1

Exponential Function

f(x) = aˣ (a>0, a≠1) e.g. 2ˣ, 3ˣ, eˣ Domain = R, Range = R⁺

Logarithmic Function

f(x) = log_a(x) Domain = R⁺ (x must be positive!) Range = R

Rational Function

f(x) = p(x)/q(x), q(x)≠0 Domain: all x except where q(x)=0

Trigonometric

sin x, cos x, tan x, etc. Period of sin,cos = 2π Period of tan,cot = π
Composition of Functions

Composite Function — gof

Apply f first, then apply g on the result. f → then → g

(gof)(x) = g(f(x)) ← read "g of f of x" Condition: Range of f ⊆ Domain of g Note: gof ≠ fog (commutative nathi in general)
📝 Example — f(x)=x+2, g(x)=x². Find gof and fog

2
(fog)(x) = f(g(x)) = f(x²) = x²+2 = x²+2
3
gof ≠ fog confirmed (x²+4x+4 ≠ x²+2)

🔰 Limit — Zero Se Samaj

Limit is about what value a function approaches as x gets very close to some number 'a'. The function may or may not be defined AT x=a, but we ask: what value does it tend to?

Limit Definition
lim f(x) = l x→a "Limit of f(x) as x approaches a equals l" Left Hand Limit (LHL): lim f(x) (x comes from left, x < a) x→a⁻ Right Hand Limit (RHL): lim f(x) (x comes from right, x > a) x→a⁺ Limit EXISTS if and only if: LHL = RHL = l

Working Rules of Limits

If lim f(x) = L and lim g(x) = M then:
lim [f(x) ± g(x)] = L ± M lim [f(x) × g(x)] = L × M lim [f(x) / g(x)] = L/M (if M ≠ 0) lim k = k (constant) lim kf(x) = kL lim [f(x)]ⁿ = Lⁿ

🔰 What is a Derivative? (Zero thi samajho)

Derivative tells us the instantaneous rate of change of a function. Geometrically, it is the slope of the tangent at any point on a curve.

Example: If s = position of a car, then ds/dt = velocity (how fast position changes). If v = velocity, then dv/dt = acceleration.

First Principles Definition
f'(x) = lim [f(x+h) − f(x)] / h h→0 Notations: f'(x), dy/dx, y', y₁, Dƒ(x) If y = f(x), then dy/dx means: "derivative of y with respect to x"

Finding Derivatives Using First Principles — 3 Examples

📝 Example 1 — d/dx(x³) using first principles
1
f(x) = x³, f(x+h) = (x+h)³ = x³+3x²h+3xh²+h³
2
f(x+h)−f(x) = 3x²h+3xh²+h³
3
[f(x+h)−f(x)]/h = 3x²+3xh+h²
4
As h→0: f'(x) = 3x²
📝 Example 2 — d/dx(eˣ) using first principles
1
f(x+h)−f(x) = eˣ⁺ʰ − eˣ = eˣ(eʰ−1)
2
Divide by h: eˣ × (eʰ−1)/h
3
As h→0: lim(eʰ−1)/h = 1 (standard limit)
4
f'(x) = eˣ × 1 = eˣ
📝 Example 3 — d/dx(sin x) using first principles
1
f(x+h)−f(x) = sin(x+h)−sin x = 2cos((2x+h)/2)·sin(h/2)
2
Divide by h: cos((2x+h)/2) × sin(h/2)/(h/2)
3
As h→0: lim sin(h/2)/(h/2) = 1, and cos((2x+0)/2) = cos x
4
f'(x) = cos x

Working Rules of Differentiation

Sum, Difference, Scalar Rules

d/dx [f ± g] = f' ± g' d/dx [kf] = k·f' (k is constant)
📝 Example — d/dx(x³ + 3x + 5)
1
= d/dx(x³) + 3·d/dx(x) + d/dx(5)
2
= 3x² + 3(1) + 0
3
= 3x² + 3

Product Rule (UV Rule)

d/dx [f × g] = f·g' + g·f' "First × deriv of Second + Second × deriv of First" For 3 functions: d/dx[fgh] = fg·h' + fh·g' + gh·f'
📝 Example — d/dx(x² · eˣ)
1
f = x², g = eˣ, f' = 2x, g' = eˣ
2
= x²·eˣ + eˣ·2x
3
= eˣ(x² + 2x) = xeˣ(x+2)

Quotient Rule

d/dx [f/g] = (g·f' − f·g') / g² "Bottom × deriv of Top − Top × deriv of Bottom, all over Bottom²"
📝 Example — d/dx(log x / x)
1
f = log x, g = x, f' = 1/x, g' = 1
2
= [x·(1/x) − log x·1] / x²
3
= (1 − log x) / x² = (1 − log x)/x²

Parametric Differentiation

When both x and y are given in terms of a third variable (parameter) t or θ:

dy/dx = (dy/dt) / (dx/dt) = g'(t) / f'(t)

What is Integration?

🔰 Integration — Reverse of Differentiation (Zero thi)

We know: d/dx(x⁵) = 5x⁴. So if someone asks "whose derivative is 5x⁴?", the answer is x⁵. Finding this "original function" is called Integration (or Antidifferentiation).

Definition
If d/dx[g(x)] = f(x), then: ∫f(x) dx = g(x) + c ∫ = integral sign (elongated S = "Sum") f(x) = integrand (function being integrated) dx = "with respect to x" c = arbitrary constant (constant of integration) Key: d/dx[∫f(x)dx] = f(x) (they are inverses)
💡
Why +c? Because derivative of any constant = 0. So x²+5, x²+100, x²−3 all have derivative 2x. When we integrate, we can't know which constant was there, so we write +c.

Definite Integral — Fundamental Theorem

Definite integral has limits (lower 'a', upper 'b'). The result is a definite number, no +c.

Fundamental Theorem of Calculus
∫[a to b] f(x)dx = [F(x)]ₐᵇ = F(b) − F(a) where F(x) = antiderivative of f(x) Note: No +c in definite integrals!
📝 Examples
1
∫[0 to 2] x² dx = [x³/3]₀² = 8/3 − 0 = 8/3
2
∫[1 to 8] 1/x dx = [log x]₁⁸ = log8 − log1 = log8 − 0 = log 8
3
∫[0 to π/2] cosx dx = [sinx]₀^(π/2) = sin(π/2) − sin0 = 1 − 0 = 1

Properties of Definite Integrals

Important Properties
P1: ∫[a→a] f(x)dx = 0 (same limits = 0) P2: ∫[a→b] f(x)dx = −∫[b→a] f(x)dx (swap limits = negative) P3: ∫[0→a] f(x)dx = ∫[0→a] f(a−x)dx (replace x with a−x) P4: ∫[a→c] f dx = ∫[a→b] f dx + ∫[b→c] f dx (split limits) P5: ∫[−a→a] f(x)dx = 2∫[0→a] f(x)dx if f is EVEN (f(−x)=f(x)) = 0 if f is ODD (f(−x)=−f(x))

What is a Logarithm?

🔰 Logarithm — Zero Thi Samajho

Logarithm is basically asking: "To what power must I raise the base 'a' to get the number N?"

Think of it as the inverse of an exponent (power). If exponent says 2³ = 8, then log says log₂ 8 = 3.

Core Definition
If aˣ = N Then x = log_a N Read as: "x is the logarithm of N to the base a" Conditions: a > 0, a ≠ 1, N > 0 Examples: 2³ = 8 ↔ log₂ 8 = 3 10² = 100 ↔ log₁₀ 100 = 2 10³ = 1000 ↔ log₁₀ 1000 = 3 3⁴ = 81 ↔ log₃ 81 = 4
💡
Easy Way to Remember: log_a N = x means "a to the power x = N". So log₂ 32 = ? means 2^? = 32. 2⁵ = 32, so answer is 5!
Types of Logarithm

Common Log vs Natural Log

Common Logarithm (log)

Base = 10. Written as log₁₀ N or simply log N.

log 1 = 0 (10⁰=1) log 10 = 1 (10¹=10) log 100 = 2 (10²=100) log 1000 = 3 (10³=1000) log 0.1 = −1 (10⁻¹=0.1)

Natural Logarithm (ln)

Base = e ≈ 2.718. Written as log_e N or ln N.

ln 1 = 0 (e⁰=1) ln e = 1 (e¹=e) ln e² = 2 ln e³ = 3 ln(1/e) = −1
Laws of Logarithm — Heart of This Chapter!

All 4 Laws — MEMORIZE These!

The Four Laws
LAW 1 — Product Law: log_a(M × N) = log_a M + log_a N (log of product = sum of logs) LAW 2 — Quotient Law: log_a(M/N) = log_a M − log_a N (log of division = difference of logs) LAW 3 — Power Law: log_a(Mⁿ) = n × log_a M (power comes in front as multiplication) LAW 4 — Change of Base: log_a b = log_c b / log_c a (convert to any other base c) Special: log_b a = 1 / log_a b
⚠️
Common Mistakes to Avoid:
❌ log(x+y) ≠ log x + log y (Correct: log(xy) = log x + log y)
❌ log(x−y) ≠ log x − log y (Correct: log(x/y) = log x − log y)
❌ (log x)² ≠ 2 log x (Correct: log(x²) = 2 log x)
📝 Examples Using Laws

2

3

4

5

6
log₃ 81 = log₃ 3⁴ = 4 log₃ 3 = 4×1 = 4

If log x = n, then finding x is called antilogarithm. Simply: x = 10ⁿ (for common log).

🔰 What is Mean? (Sauthi Basic)

Mean = Sum of all values / Total number of values. It is the "average" of a data set.

Mean for Ungrouped Data
x̄ = x₁+x₂+...+xₙn = Σxᵢn For grouped data (with frequencies f): x̄ = ΣfᵢxᵢΣfᵢ

1
Sum = 3+5+2+6+5 = 21
2
n = 5
3

📝 Example 2 — Mean of first 5 natural numbers (1,2,3,4,5)
1
Sum = 1+2+3+4+5 = 15
2
n = 5
3
x̄ = 15/5 = 3

Method 2 — Assumed Mean Method

Use when data values are large. Choose any convenient assumed mean 'a' (usually middle value).

x̄ = a + (Σfᵢdᵢ / Σfᵢ) where: a = assumed mean (chosen by us) dᵢ = xᵢ − a (deviation from assumed mean) fᵢ = frequency

Method 3 — Step Deviation Method

Most efficient for large data with equal class widths.

x̄ = a + (Σfᵢuᵢ / Σfᵢ) × h where: a = assumed mean h = class width (class interval) uᵢ = (xᵢ − a)/h (step deviation)
Median

Median — The Middle Value

Median is the middle value when data is arranged in increasing or decreasing order.

Median for Ungrouped Data
Arrange data in order. If n is ODD: Median = ((n+1)/2)th term If n is EVEN: Median = average of (n/2)th and (n/2+1)th terms
📝 Example — Median of 3, 7, 1, 9, 4
1
Arrange: 1, 3, 4, 7, 9
2
n=5 (odd), middle = (5+1)/2 = 3rd term
3
Median = 4
Median for Grouped Data (Frequency Table)
M = l + [(n/2 − cf)/f] × h l = lower class boundary of MEDIAN CLASS n = total frequency (Σf) cf = cumulative frequency BEFORE median class f = frequency OF median class h = class width Finding Median Class: → Find n/2 → Find class whose cumulative frequency FIRST EXCEEDS n/2 → That is the median class
Mode

Mode — Most Frequent Value

Mode = value that appears maximum number of times in the data.

Mode for Grouped Data
Z = l + [(f₁ − f₀)/(2f₁ − f₀ − f₂)] × h l = lower boundary of MODAL CLASS (class with highest freq) f₁ = frequency of modal class (highest frequency) f₀ = frequency of class BEFORE modal class f₂ = frequency of class AFTER modal class h = class width
Empirical Relationship: Mode = 3 × Median − 2 × Mean (approximate relation for moderately skewed data)
Mean Deviation

Mean Deviation from Mean

Mean deviation measures the average distance of data points from the mean.

Mean Deviation Formulas
For ungrouped data: MD = (1/n) × Σ|xᵢ − x̄| For frequency distribution: MD = Σfᵢ|xᵢ − x̄| / Σfᵢ
📝 Example — Find mean deviation of 2, 4, 7, 8, 9
1
Mean x̄ = (2+4+7+8+9)/5 = 30/5 = 6
2
|2−6|=4, |4−6|=2, |7−6|=1, |8−6|=2, |9−6|=3
3
Sum = 4+2+1+2+3 = 12
4
MD = 12/5 = 2.4
